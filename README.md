## Features

-  **PDF Editor** — Add text, images, links & freehand drawings
-  **Compressor** — Reduce PDF file size
-  **Image to PDF** — Convert images to PDF instantly
-  **Merge PDF** — Combine multiple PDFs into one
-  **Split PDF** — Extract pages from a PDF
-  **BG Remove** — Remove image backgrounds
-  **QR Tools** — Generate & scan QR codes
-  **Resume Builder** — Clean resume in seconds

## Run Locally

```bash

cd pdfkholo
npm install
npm run dev
```
